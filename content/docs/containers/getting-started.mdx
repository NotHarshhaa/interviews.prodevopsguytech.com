# Getting Started

<Callout type="info" title="Overview">
Welcome to our comprehensive guide on Container technologies interview questions. This guide covers everything from basic concepts to advanced implementation strategies in Docker and Kubernetes.
</Callout>

## Question Categories

<Callout type="tip" title="Learning Path">
Start with Docker basics before moving to Kubernetes. Understanding container fundamentals is crucial before diving into orchestration.
</Callout>

### Beginner Level Questions
<InteractiveHoverButton text="Go to Beginner Level" href="./beginner" className="mt-2" />

Fundamental concepts including:
- Docker Basics and Architecture
- Container Lifecycle Management
- Basic Docker Commands
- Kubernetes Fundamentals
- Pod and Deployment Basics

### Intermediate Level Questions
<InteractiveHoverButton text="Go to Intermediate Level" href="./intermediate" className="mt-2" />

Advanced concepts including:
- Docker Image Optimization
- Container Networking
- Docker Compose
- Kubernetes Services and Ingress
- StatefulSets and Volumes

### Advanced Level Questions
<InteractiveHoverButton text="Go to Advanced Level" href="./advanced" className="mt-2" />

Expert topics including:
- Container Security
- Advanced Docker Features
- Kubernetes Architecture
- Custom Controllers
- Advanced Deployment Strategies

<Callout type="warn" title="Important">
Container security is crucial! Always follow security best practices, use trusted base images, and regularly scan for vulnerabilities in your container images.
</Callout>
